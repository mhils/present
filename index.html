Introduction

What is this framework about?

  • Based on a fork of impress.js. You should skim the docs.
  • Bootstrap & FontAwesome included, Lato as font.
  • CSS to support "print to pdf"
    (always needs adjustments, but mostly works)
  • Optional live JS editor using CodeMirror
  • Optional LaTeX support using MathJax
  • Some nice additions: progress bar, page numbers, skippable slides, substeps, spoilers, ...
  • Focus: No bloat, just some very light modules.
